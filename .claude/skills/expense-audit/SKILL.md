# expense-audit

Скилл для аудита расходов из `.xlsx`-файла на Google Drive.
Скрипт находится в `/Users/nikolass/datapeople/expense-audit/audit.py`.

Аргументы при вызове: `$ARGUMENTS`

---

## Шаг 1 — Получение ссылки

Если `$ARGUMENTS` содержит URL Google Drive — используй его напрямую.

Если аргументов нет — спроси пользователя:
> «Вставь ссылку на файл расходов в Google Drive (`.xlsx`).»

Принимаемые форматы:
- `https://drive.google.com/file/d/FILE_ID/view`
- `https://docs.google.com/spreadsheets/d/FILE_ID/edit`
- `https://drive.google.com/open?id=FILE_ID`
- Сырой `FILE_ID` (строка длиной 20+ символов)

---

## Шаг 2 — Запуск аудита

```bash
cd /Users/nikolass/datapeople/expense-audit
python3 audit.py --file-url "<ССЫЛКА>"
```

Скрипт сам:
1. Извлекает `FILE_ID` из ссылки
2. Скачивает файл через `gws drive files get`
3. Читает первый лист, автоопределяет колонки (рус/англ):
   `дата/date`, `сумма/amount`, `категория/category`, `описание/description`
4. Запускает аудит
5. Добавляет (или перезаписывает) лист `Audit Report`
6. Загружает файл обратно на Google Drive

---

## Шаг 3 — Интерпретация результатов

После успешного завершения скрипт выводит в консоль сводку. Прочитай её и
представь пользователю в удобном виде:

**Что выделить:**
- Общая сумма расходов и количество строк
- Топ-3 категории по сумме (с процентами)
- Количество аномалий (если > 0 — перечисли категории с аномалиями)
- Количество дублей (если > 0 — назови даты/суммы)
- Тренд: вырос или снизился расход в последнем месяце по сравнению с предыдущим

**Пример итогового сообщения:**

```
Аудит завершён ✅
Прочитано строк: 143 | Сумма: 487 320 ₽

Топ категории:
  Аренда       180 000 ₽  (36.9%)
  Зарплаты     120 000 ₽  (24.6%)
  Маркетинг     58 500 ₽  (12.0%)

⚠️  Аномалий: 2 (Маркетинг, Прочее)
🔁 Дубликатов: 0
📈 Март → Апрель: +12 400 ₽ (+2.6%)

Лист «Audit Report» добавлен в файл на Google Drive.
```

## Обработка ошибок

| Ошибка | Действие |
|---|---|
| `Cannot extract file ID from URL` | Попроси пользователя вставить другую ссылку или чистый ID файла |
| `Download failed` | Проверь, что файл доступен по ссылке и не требует запроса доступа |
| `Could not find an 'amount' column` | Спроси пользователя, как называется колонка с суммой в его файле, и передай через `--amount-col` (или объясни, что нужно переименовать колонку) |
| `Upload failed` | Проверь права на запись: у аккаунта `gws` должно быть право редактирования файла |
| Файл не `.xlsx` (Google Sheets) | Сообщи пользователю, что нужно скачать файл как `.xlsx`: Файл → Скачать → Microsoft Excel |
